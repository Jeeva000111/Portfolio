VR Dental Training Simulation

Virtual RealityTrainingUnreal Engine

Interactive VR training using real patient dental scans. Key procedures are simulated with real‑time tool interactions, detailed 3D assets, dynamic lighting and materials—creating a repeatable, safe environment for understanding techniques.

Client: Incise Dental
Software: Unreal Engine
Type: Virtual Reality
Goal: Enable immersive, hands‑on dental training using real patient models in VR.

My role

Scan cleanup/retopo; tool interactions; scenario steps & evaluation; lighting & materials.

Tools / Stack

Unreal Engine, asset pipeline

Context

Freelance training product for patient‑model based education.

Process

  1. Discovery & Planning: Defined project scope, requirements, and constraints in consultation with stakeholders.
  2. Design: Created wireframes, 3D models, and interaction flow to ensure optimal user experience and technical feasibility.
  3. Development: Implemented the environment, interactions, and assets using Unreal Engine and relevant toolchains.
  4. Testing & Iteration: Conducted multiple test runs, collected feedback, and iteratively refined features.
  5. Deployment: Packaged and delivered the project for live demonstration, ensuring stability and performance.

Process

  • Convert CT/CBCT scans to clean meshes suitable for real‑time use.
  • Assemble a UE scene with accurate scale, lighting, and materials.
  • Build dental tools/interactions in Blueprints with constraints for stable control.
  • Script procedure steps with checks, timers, and scoring to measure performance.
  • Iterate via user tests; optimize and produce a walkthrough video.
