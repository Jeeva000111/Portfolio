Maharashtra Drone Mission — VR Command Truck

Virtual RealityUnreal EngineBlender

This project focused on creating a fully functional and immersive VR simulation of a mobile command truck. Designed as a centralized War Room, it supports real‑time monitoring, coordination, and response planning across Maharashtra. Users explore the truck in VR, engage with operational interfaces, and simulate decision‑making to enhance situational awareness and training.

Client: Government of Maharashtra
Software: Unreal Engine, Blender
Type: Virtual Reality
Goal: Enabled immersive, real‑time environmental exploration through a fully interactive VR experience.

My role

Environment, interaction and camera systems; interior/exterior modeling support; VR performance and UX polish.

Tools / Stack

Unreal Engine 5.3 (Blueprints, Sequencer), Blender, spatial audio

Context

Centralized ‘War Room’ concept for response planning in emergencies.

Process

  • Collect ops requirements for the War‑Room truck; define user tasks and viewpoints.
  • Block out truck interior/exterior from references; plan operator layout and screen positions.
  • Set up UE project for VR: lighting, materials, target frame rate, input, and locomotion.
  • Blueprint interaction widgets for screen feeds, toggles, and perspective switching.
  • Author spline‑based vehicle movement; build multi‑camera rig and cut rules.
  • Synchronize spatial audio and lights with motion/events for situational cues.
  • Profile, optimize (LODs/materials), user‑test in headset, and record the demo.

Process

  1. Discovery & Planning: Defined project scope, requirements, and constraints in consultation with stakeholders.
  2. Design: Created wireframes, 3D models, and interaction flow to ensure optimal user experience and technical feasibility.
  3. Development: Implemented the environment, interactions, and assets using Unreal Engine and relevant toolchains.
  4. Testing & Iteration: Conducted multiple test runs, collected feedback, and iteratively refined features.
  5. Deployment: Packaged and delivered the project for live demonstration, ensuring stability and performance.
